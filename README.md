Tela de login simples feita durante estudos com ajuda do Curso em Vídeo.

O projeto consiste no desenvolvimento de uma interface de login utilizando HTML e CSS, com foco em organização visual, usabilidade e responsividade. A página aplica conceitos como Mobile First, centralização de elementos, uso de ícones e padronização de cores para criar uma experiência simples e intuitiva ao usuário. O layout foi pensado para se adaptar a diferentes tamanhos de tela.

A página pode ser visulizada em: https://naeorii.github.io/tela-login/
